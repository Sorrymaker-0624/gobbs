# 用户模块 API

## 用户注册

* **功能描述**: 创建一个新的用户账户。
* **URL**: `/register`
* **请求方法**: `POST`
* **请求参数 (form-data)**:
  | 参数名      | 类型     | 是否必须 | 描述             |
  | :---------- | :------- | :------- | :--------------- |
  | `username`  | `string` | 是       | 用户名           |
  | `password`  | `string` | 是       | 密码             |
  | `email`     | `string` | 否       | 电子邮箱         |

* **成功响应**:
    ```json
    {
        "code": 0,
        "message": "注册成功"
    }
    ```

* **失败响应 (示例)**:
    ```json
    {
        "code": 1001,
        "message": "用户名已存在"
    }
    ```
